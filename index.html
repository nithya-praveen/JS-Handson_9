Q7. Find the sum of Object Members You are given a function Check which takes an object Obj as a parameter.Your taks is to find out the sum of the three data members of the object. The data members are named as p1, p2, p3.

A7. function Check(obj1)
{
let sum = 0;
for (const value of Object.values(obj1))
{
sum += value;
}
return sum;
}
